## E-Commerce-Shopping Using Python

A Python terminal-based E-Commerce application where users can browse categories, view products, add multiple items to cart, calculate GST, make payments, and generate receipts. Choose payment methods (PhonePe, Google Pay, Card, or Cash on Delivery), verify with UPI PIN or card password, and generate a final receipt.
This project demonstrates core Python concepts, login system, billing, and payment flow in a console environment.

##  Key Features

* User & Admin login system
* Multiple products with different GST for each category
* Multiple payment methods (UPI, Card, Cash on Delivery)
* Check User Balance
* Receipt generation with full bill details
